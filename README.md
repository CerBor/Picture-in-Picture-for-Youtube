## How to Install:
1. Code > "Download Zip".
2. Unpack the ```src``` folder to a convenient location.
3. "Additional tools" > "Extensions".
4. Turn on "Developer Mode".
5. Press "Load unpacked extension".
6. Select the path to the ```src``` folder.
